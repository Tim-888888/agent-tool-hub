Create `src/app/api/tools/[slug]/tags/route.ts` with two handlers:

**GET handler** -- Get tag vote counts for a tool:
1. Look up tool by slug. If not found, return 404.
2. Use `prisma.toolTagVote.groupBy` to get count per tagSlug:
   ```typescript
   const tagCounts = await prisma.toolTagVote.groupBy({
     by: ["tagSlug"],
     where: { toolId: tool.id },
     _count: { tagSlug: true },
     orderBy: { _count: { tagSlug: "desc" } },
   })
   ```
3. Return `{ success: true, data: tagCounts.map(t => ({ tagSlug: t.tagSlug, count: t._count.tagSlug })) }`

**POST handler** -- Vote or unvote a tag (per D-19 -- toggle, max 3):
1. Call `requireAuth()` -- if error, return 401
2. Parse request body with Zod: `{ tagSlug: z.string() }`
3. Validate tagSlug exists in TAG_PRESETS (per D-23 -- preset only). If invalid, return 400.
4. Look up tool by slug. If not found, return 404.
5. Check if user already voted for this tag:
   ```typescript
   const existing = await prisma.toolTagVote.findUnique({
     where: { toolId_tagSlug_userId: { toolId: tool.id, tagSlug, userId: session.user.id } }
   })
   ```
6. If existing: delete the vote (unvote), return `{ success: true, data: { action: "unvoted", tagSlug } }`
7. If not existing:
   - Check vote count (per D-19, max 3):
     ```typescript
     const voteCount = await prisma.toolTagVote.count({
       where: { toolId: tool.id, userId: session.user.id }
     })
     if (voteCount >= 3) return errorResponse("Max 3 tags per tool", 400)
     ```
   - Create the vote:
     ```typescript
     await prisma.toolTagVote.create({
       data: { toolId: tool.id, tagSlug, userId: session.user.id }
     })
     ```
   - Return `{ success: true, data: { action: "voted", tagSlug } }`

Follow existing route handler pattern: `await params` for dynamic segments, try/catch, errorResponse helper.
